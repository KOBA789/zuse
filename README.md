# Zuse

Relay Switch Circuit CAD & Simulator

Live demo: https://zuse.koba789.com/

## Build & Development

### Requisites

- Node.js >= v14
- yarn v1
- Rust >= 1.56
- wasm-pack >= 0.10

### Debug build

Use `yarn dev`. Vite dev server spins up.

### Release build

Use `yarn build`. It puts artifacts in `dist/`.
